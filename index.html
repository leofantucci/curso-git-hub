Mini Curso de Git e GitHub

Professor Miguel e Professora Raissa

23 - 10 - 2025

No mini-curso oferecido pelos alunos Miguel Totti e Raissa Santana foi possivel aprender o básico sobre a tecnologia Git, e o repostório online GitHub.
Aprendemos a seguir os seguintes passos para subir seu código online:

→ Crie uma pasta em seu computador no diretório desejado (é possivel usar mkdir no CMD para criar).

→ Crie o código desejado com as alterações necessárias.

→ Após isso, abra o Git Bash/Terminal e dê o comando 'git init'.

→ Agora você precisa criar um repositório remoto, um local origem onde você pode colocar seus códigos. Para isso, use: 'git remote add origin https://github.com/' + 'nomeDoSeuPerfil' + '/' + 'nomeDaSuaPasta'.

→ Podemos então, criar uma ramificaçãp principal, com o comando git branch -M main.

→ Com isso, você pode ir adicionando as alterações feitas em suas versões do git através do comando: git add . para adicionar tudo na pasta ao git.

→ Quando você tiver adicionado tudo que é desejado ao seu git, você pode dar um commit, que é nada mais nada menos que adicionar seu código à linha principal (MAIN) do seu repositório git commit -m "numero do commit".
